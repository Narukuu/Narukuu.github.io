linux系统的基本操作

#Linux 必备系统命令

ls ->显示当前目录下的所有文件

ls-l 列表形式展示文件

ls-a 显示所有的文件（包括隐藏文件）

ls-h 按字节的形式展示文件大小

mkdir ->创建目录 mkdir +文件名字

mkdir-p 递归创建子目录

例 mkdir 1/2/3/4-p
如何判读已经创建
find 1/
pwd 显示当前位置

touch ->创建文件 vim也可以创建文件 i编辑 在文本中查找关键字 ：/要查找的字符串
1.cat 读文件 cat-n 标上行号 tac 倒序排序 cat 一页一页查找文件
2.more 显示查找文件 逐步展示 less也差不多 不过是逐行展示
3.head 查看头几行 例如 head -3 2.txt 展示前三行 tail 倒序
4.cp 1.txt 3.txt 将文件1复制到文件2
复制到文件夹 ：cp 1.txt 2/ 将文件1复制到文件夹2中
5.剪切命令：例如将1.php剪切到文件夹2中 ->mv 1.php 2/ 剪切和cp的用法是差不多的
还可以用来重命名 例如将 1.py 改成 2.py ->mv 1.py 2.py
6.删除命令：rm +(要删除的文件) y 删除 n 不删除
rm-f 不用提示是否要删除
如何删除一个目录->无法直接删除（带有其他文件）
rm -rf 递归删除 rm-r 2/ ->递归删除整个目录 rm-rf ->不带提示删除整个目录
此命令比较危险，要注意使用。rm-rf/* 删除跑库 删除所有文件
7.文件搜索命令：find 查找文件或目录
find<搜索范围><匹配条件>

find -name 按文件名搜索
例：find -name 1.txt 直接简单粗暴查找
如果是查找一种类别的文件的话 find -name “*.txt”

a是读取，c是修改
find -amin-n 在过去n分钟内被读取过的文件
find -atime 在过去n天内被读取过的文件
find -anewer file :比文件file更晚被读取过的文件
find -cmin-n 在过去n分钟内被修改过的文件
find -cniewer file 比文件file 更新的
find -ctime -n 在过去n天内被修改过的文件

按类型
find -type f 文件 查询当前文件夹类有哪些文件
find -type d 目录
find -type l 链接

查找系统中所有文件长度为0的普通文件 并列出他们的完整路径
find /-type f -size -exec ls -l {} ;
按 CTRL c 结束

权限管理命令
chmod 更改文件权限

chmod +x 文件目录
chomd -R u+rwx 文件/文件夹
chomd -R g+rwx 文件/文件夹
chomd -R o+rwx 文件/文件夹

用户权限查看
查看属性 drwxr ->d->文件夹 l->链接 ”-“->普通文件 c->字符串
三个字母连着一起看 第一组代表所有者用户权限 第二组->该文件所在组的用户所运用的权限
r读 w写 x执行
例如 给文件夹1赋上x权限
chmod +x 1/
例如 单独给其中一个用户更改文件权限 u->所有者 g所在组的用户 o->其他用户

授予用户，组、其他人对文件或文件夹拥有rwx权限
chomd -R +777 文件/文件夹
疑惑：为什么777代表可读可写可执行
二进制权限(1代表有其权限 0代表无权限)
rwx=111=4+2+1=7
r-x=101=4+0+1=5
rw-=110=4+4+0
r–=100=4+0+0=4
（也不是很难啊）

压缩解压命令
gzip:（不保留源文件）压缩和解压缩.gz文件
gzip<文件>#压缩文件
gunzip/gzip -d<文件> #解压

zip –>压缩和解压缩zip文件（可压缩文件夹）（保留源文件）
zip -r<压缩后文件名><目录>
unzip<zip压缩文件>

tar —>打包和压缩目录（保留源文件）
tar->zcf<压缩后文件；名><被压缩目录>#打包
tar->zxvf 压缩包文件名 #解压
-f 制定文件名
-v 显示详细信息
-z 打包/解包的同时压缩/解压（要放在选项的前面）
例如： tar -zcf 2.tar.gz 2.txt
将2.txt 打包压缩为2.tar.gz

网络命令

  1. ping :测试网络连通性
  2. ifconfig: 查看当前的网络配置信息
  3. netstat: 显示网络相关信息
    netstat -tlun 查看本机监听的端口
    netstat -rn 查看本机路由表
    netstat -anp 列出所有端口信息，最后一列为进程ID
    netstat -anpt 查看tcp端口信息

其他命令
uname -a 查看当前操作系统的linux内核信息
（经常会根据内核版本来判断是否有本地提权的风险）
whoami :查看当前用户
id:显示当前已登陆的用户的组
last:显示之前用户登录的信息
lastlog:显示所有用户最后一次登录的信息
useradd:添加用户
userdel:删除用户
直接userdel test10 会不行 test10在创建后就会有进程占用
ps
关闭–> kill -9 +用户相对应的PID 关闭进程
userdel test10
但在home 文件夹还是会保留有test10文件夹

passwd:修改密码 输入新密码即可（更改root）
如果想修改创建的新用户的密码 (创建的新用户的名称叫test10)
->su root
->passwd test10

cat /var /log :查看日志目录 按两次tab键
history:查看历史操作命令
history-c：删除历史操作命令
cat~/.bash_history#查看历史操作命令 （上述删除后还是可以通过这个来查看）
crontab–计划任务
crontab-l –>列出系统的计划任务
crontab-e –>进入编辑模式
crontab -u root -e –>查看某个特定用户

查看进程
ps —查看系统进程
ps aux:列出全部进程
a–:查看其他用户的进程
x–:查看自己的进程
u–:显示进程的用户名和时间
#查看关于nginx的相关进程信息
ps aux | grep nginx 查看某个进程

grep 查找
ps -u 用户 指定进程用户查找
#查看指定用户的相关信息
ps -u root

top—–>是一个实时刷新的进程信息
c:显示进程的完整命令
p:CPU降序的方式排序
M:按内存的占用大小来排序
q:退出
Ctrl+c：退出
